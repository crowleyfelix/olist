# Work at Olist

This project is an API that provide endpoints to record and calculate telephone calls.

## Installing

### Prerequisites
- [docker]
- [docker-compose]


Execute the follow command to start the application:

`docker-compose up -d`

To stop, execute:

`docker-compose down`
